Criminal Victimization, 2017       NCJ 252472	
	
This zip archive contains tables in individual  .csv spreadsheets	
from Criminal Victimization, 2017,  NCJ 252472.  The full report including text	
and graphics in pdf format is available from: https://www.bjs.gov/index.cfm?ty=pbdetail&iid=6466
	
This report is one in a series.  More recent editions	
may be available.  To view a list of all in the series go to	
https://www.bjs.gov/index.cfm?ty=pbse&sid=6
